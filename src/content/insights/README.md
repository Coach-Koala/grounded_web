# Insights content

One `.md` (or `.mdx`) file per post. Required frontmatter — the build fails
loud if any field is missing:

```markdown
---
title: "Why your stop-loss renewal quote is negotiable"
date: "2026-08-01"
summary: "One-paragraph teaser shown on the index page."
---

Body in markdown. Voice rules apply: lead with the number, short sentences,
cite sources (KFF, Mercer, Health Affairs), no pricing numerals for Grounded
offers, never "HippoHealth".
```

The filename becomes the URL slug: `stop-loss-renewals.md` →
`/insights/stop-loss-renewals/`.
